# 主世界规则（地球穿越时代·主世界设定）

## 1. 世界背景与起源
地球（蓝星）原本是普通的科技文明星球，无超凡力量。一夜之间爆发不可逆的天地异变，海量幻想异界、衍生位面强行拼接锚定地球，彻底改写法则、生态与文明。

异变后地球持续膨胀扩张，最终成长为高阶世界。人类凭借精神力沟通新生超凡法则，开启异界通道，进入各类衍生世界历练掠夺、锤炼自身。全球进入“超凡至上”的新纪元，科技沦为辅助。

## 2. 法则化（觉醒核心）
- **条件**：年满18周岁，最多3次机会；30周岁后永久失去资格。
- **过程**：通过“世界之舟”进入专属法则化试炼世界，完成随机任务（猎杀/碰触型）。试炼期间**严禁**携带任何超凡道具、高科技物品，否则直接失败。
- **成功率**：整体不足10%，失败者成为半觉醒者。
- **收益**：身体完成法则化，获得客观属性面板、可承载超凡力量、可进入异界、免疫世俗疾病、寿命大幅延长。
- **完成度**：决定职业星级、天赋潜力、后续成长上限。

**特殊说明**：主角季小忘（第一人称“我”）因特殊原因可能触发双职业（详见 character-cards/季小忘-panel.md）。

## 3. 数据化身体与个人面板
面板为法则化后的客观呈现，无任何自动加点、任务系统、一键领取功能。所有成长靠实战、资源、领悟。
- 包含：姓名、等级、职业、基础属性（力敏体精智）、技能、装备、领地、任务、声望等。
- 每升一级获得自由属性点，不同职业有专属加成倾向。

## 4. 阶位与晋升
- 过渡期：初级/中级/高级觉醒者（未入正式1阶）
- 正式阶位：1-9阶（每阶分初期/中期/后期/巅峰）
- 晋升：纯实战 + 资源 + 法则领悟 + 进阶试炼，无捷径。

## 5. 技能体系
- 获取：实战击杀、秘境传承、技能晶石、自创。
- 分类：主动/被动/天赋。
- 升级：基础技能靠苦练，战斗技能可用晶币+熟练度，高阶需专属材料。
- 融合：同系技能可融合为更强复合技能。

## 6. 领地建设系统
- 核心道具：**城镇之心**（对应世界阶位）。
- 建立位置：衍生世界。
- 升级路径：初级村落 → 小镇 → 城池 → 大城 → 王城 → 秘境领地。
- 功能：招募兵种/英雄、生产资源、提供法则加持、反哺领主修炼。
- 规则：私人领地受法则绝对保护；领主死亡则领地崩塌。

## 7. 资源与经济系统（详见 economic-system.md）
- 货币：白晶币（低阶）→ 红晶币 → 黄晶币 → 黑晶币（顶级，不可交易）。
- 获取：异界怪物掉落、任务奖励、领地产出、交易。
- 装备/材料稀有度：普通→优秀→精良→史诗→传说→神话。

## 8. 穿越机制（主世界 → 衍生世界）
- 方式：固定通道、坐标道具、城镇之心、随机献祭。
- 法则融合：进入后主世界法则部分生效，实力可能被世界等级压制。
- 时间流速：未占领衍生世界流速极快（1年≈主世界1分钟）；已占领则同步。
- 返回：完成任务/死亡（低阶）/返回卷轴；有24小时冷却，同一世界7天间隔。
- 身份伪装：世界法则自动生成符合当地背景的身份。

## 9. 其他核心规则
- **社会阶层**：觉醒者（统治阶级）> 半觉醒者 > 普通人。
- **势力**：五大联盟 + 中立势力 + 敌对势力（详见后续补充）。
- **世界意志**：高阶（9阶）本土觉醒者可能被控制（主角季小忘例外）。
- **怪物/BOSS/秘境**：主要在衍生世界，本土危险区/禁地极度危险。
